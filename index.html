CSS Positioning

Specify particular margins in css with these commands

  • margin-top: /*some value*/
  • margin-right: /*some value*/
  • margin-bottom: /*some value*/
  • margin-left: /*some value*/

Specify an element's margins all at the same time, starting from top margin and go clockwise, from top to right to bottom to left.

margin: 1px 5px 1px 5px;

This example sets a top margin of 1 pixel, a right margin of 5 pixels, a bottom of 1 pixel, and a left of 5 pixels.

The border is currently commented out, so you won't see the shift of the div. If you go to the stylesheet, uncomment out the line of code, you can see how the element is affected.

Padding

Padding can be set two different ways, similar to margins. They can be set individually or in one declaration.

  • padding-top: /*some value*/
  • padding-right: /*some value*/
  • padding-bottom: /*some value*/
  • padding-left: /*some value*/
padding: value value value value;

If the padding will be the same for all the sides, it can be declared like so

padding: 10px;
